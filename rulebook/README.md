# Rulebook

Правила игры для системы **SpaceHolder**, изложенные от лица игрока и ведущего,
а не от лица кода. Тексты в этой папке должны читаться как главы из книги
правил: формулировать механику последовательно, давать термины и примеры,
избегать программистского жаргона.

> Источник истины по реализации остаётся в `module/**`. Если правило в этой
> папке расходится с кодом — это баг документации или кода; сверяться и
> синхронизировать.

Список багов из какого-то теста с игроками средней свежести. Некоторые могли быть пофикшены
- Нет пермишшена на удаление и иногда изменение сообщения
- При реакции игрока она не происходит
- Добавить удаление кучи
- Убрать надеть если нельзя надеть
- Игрок не может апдейтнуть чужой токен чтобы создать травму
- У верхней травмы перекрывается инфобокс источника
- При смене стороны после начала боя но до выбора того кто ходит остаётся выбор токена который сменил сторону
- После завершения боя ОД должны восстанавливаться до максимума.

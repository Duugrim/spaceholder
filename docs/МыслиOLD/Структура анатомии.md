# Основная идея
У каждого токена (персонажа, нпц, животного, машины, и т.д.) есть анатомия, представляющая собой иерархическое дерево вида:

- Основная часть
	- Тоже часть
	- Ещё часть
		- Часть ещё части
		- Тоже часть ещё части
			- Сабчасть тоже части ещё части

И так далее.

**Основная часть** - часть, в которую больше всего вероятность попасть. К ней коннектятся другие части, а к ним - ещё, и так далее.

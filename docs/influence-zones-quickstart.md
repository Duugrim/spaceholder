# Быстрый старт: Сферы влияния

## Что это?

Система позволяет создавать объекты со сферами влияния (как территории в Stellaris). Объекты одной стороны объединяются в общую территорию.

## Как использовать?

### 1. Создайте Global Object

- Откройте Actors Directory
- Создайте нового актора, выберите тип **Global Object**
- Настройте параметры:
  - **Range** — радиус влияния (в клетках)
  - **Power** — сила
  - **Faction** — UUID Journal фракции (JournalEntry/JournalEntryPage)

### 2. Разместите на карте

- Перетащите Global Object на сцену
- Создайте несколько объектов с разными параметрами
- Объекты с одинаковой Faction будут одного цвета

### 3. Визуализируйте зоны

Откройте консоль (F12) и выполните:

```js
game.spaceholder.showInfluence()
```

Чтобы скрыть:
```js
game.spaceholder.hideInfluence()
```

## Цвета

- Цвет берётся из папки Journal фракции (если задан цвет папки)
- Иначе используется детерминированный fallback-цвет

## Советы

- Используйте Range 100-200 для нормального размера
- Объекты одной стороны визуально сливаются
- Можно создавать сотни объектов, но визуализация будет по команде
- Пересчёт НЕ автоматический — нужно вызывать функцию заново

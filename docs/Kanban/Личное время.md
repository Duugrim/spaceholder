# Личное время

Инфраструктура персональных секунд актёра (без потребителей DoT).

## Сделано

- Конверсия: `seconds = spentAp * 10 / maxAp` (полный пул ≈ 10 с)
- Тик при `spendAp` и при `refreshApPool` (остаток ОД до refill)
- GM Token Controls → «Пропустить время» (абсолютные сек/мин, без траты ОД)
- Хук `spaceholder.personalTimeAdvanced`, флаг `flags.spaceholder.personalTime.totalSeconds`
- Док: `docs/code/reference/PERSONAL_TIME.md`

## Проверить руками

1. Вне боя: потратить ОД действием → `getPersonalTimeTotal(actor)` вырос на `ap * 10 / max`.
2. В бою: начать ход с ненулевым остатком ОД → время выросло на остаток; пул полный.
3. Undo траты ОД → totalSeconds откатился.
4. ГМ: Token Controls → Пропустить время → выбранные / сцена / мир; ОД не меняются, часы двигаются.
5. В консоли: `Hooks.on("spaceholder.personalTimeAdvanced", (a, p) => console.log(a.name, p))`.

# Снаряд

Блок «что доставляется в цель»: либо у оружия (`weapon.<канал>.projectile`), либо у боеприпаса (`weapon.ammo.projectile`). В рантайме для дальнобойного выстрела с внешним патроном итоговый объект снаряда собирается как **база с оружия + поверх неё поля с патрона, если на патроне они «осмысленны»** (положительный урон, непустой тип урона, положительные AP/ADF/твёрдость, непустой `payloadId`, непустой `applications` / `builderId` и т.д. — см. `_mergeProjectile` в `module/helpers/item-nested-storage.mjs`). То есть оружие может задавать запасной/дефолтный снаряд, а патрон — перекрывать ключевые величины.

Пакет урона для брони/анатомии строится из снаряда через `composeProjectileApplications` / `buildProjectileApplications` (`module/documents/item.mjs`): либо явные **фазы** `applications` (последовательно или параллельно, несколько типов урона), либо легаси-одиночка `damage` + `damageType` (+ те же модификаторы брони по умолчанию на весь удар).
